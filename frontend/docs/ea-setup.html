Installer l'Expert Advisor MT5

L'EA est un petit programme MQL5 qui tourne dans ton terminal MT5 et exécute les setups générés par Scalping Radar. 5 minutes pour tout configurer, fonctionne sur Windows et Mac (MT5 a une version Mac officielle Pepperstone).

00Prérequis

  • MT5 Desktop installé et connecté à ton compte broker — compte démo recommandé pour les premiers tests
  • Un compte Premium sur Scalping Radar (voir les plans)
  • ~5 minutes
Pas besoin de
Python, Tailscale, VPS Windows, ou pair-RDP. Tout se passe dans MT5 directement.

MSur Mac — différences à connaître

Si tu es sur Mac, la procédure est identique à 95 %, mais quelques points diffèrent. Lis cette section avant de commencer, puis suis les étapes 01 → 06 normalement.

Installer MT5 Mac

  • Télécharge MT5 Mac depuis le site Pepperstone : pepperstone.com/.../metatrader-5 (bouton "Download for Mac"), ou directement chez MetaQuotes : metatrader5.com/.../download
  • Le .dmg contient un installer auto-suffisant (Wine intégré géré par MetaQuotes — pas besoin d'installer Wine séparément)
  • Apple Silicon (M1/M2/M3/M4) supporté nativement depuis 2023, pas besoin de Rosetta
  • Première ouverture : clic droit sur l'icône MT5 dans Applications → "Ouvrir" (macOS demande confirmation pour app non App Store)
  • À la 1ère connexion réseau, macOS demande "MT5 souhaite accepter les connexions entrantes" → autoriser, sinon WebRequest ne fonctionnera pas

Différences de menus

Étape Windows Mac
Options / Settings Tools → Options MetaTrader 5 → Settings (menu app, en haut à gauche de l'écran)
MetaEditor Tools → MetaQuotes Language Editor ou F4 F4 (le menu Tools n'existe pas en haut, il est dans le menu app)
AutoTrading toggle Ctrl+E ou bouton barre d'outils ⌘+E ou bouton barre d'outils
Properties EA F7 sur le chart F7 sur le chart (identique)

Data Folder Mac

Quand tu fais File → Open Data Folder, MT5 ouvre Finder à l'emplacement correct, qui ressemble à : 

~/Library/Application Support/net.metaquotes.wine.metatrader5/drive_c/Program Files/MetaTrader 5/

Tu n'as pas à connaître ce chemin par cœur — laisse MT5 l'ouvrir pour toi via le menu. Navigue ensuite dans MQL5/Experts/ et glisse-dépose ScalpingRadarEA.mq5 depuis Téléchargements.

Astuce Mac
Ne double-clique pas sur le .mq5 dans le Finder — macOS va essayer de l'ouvrir avec un éditeur de texte au lieu de le déposer dans MT5. Drag & drop uniquement.

Pièges spécifiques Mac

  • WebRequest qui timeout silencieusement : sur Wine, le DNS Mac peut ne pas se propager correctement. Si tu vois fail > 0 dans le log alive, ouvre Terminal et tape ping app.scalping-radar.online. Si ping OK mais EA fail → redémarre MT5 (souvent ça suffit)
  • MT5 freeze à l'ouverture : signe d'install Wine corrompue. Drag MT5 vers Corbeille puis réinstalle (5 min, résout 95 % des cas)
  • Drag & drop dans Navigator MT5 ne marche pas : après avoir déposé le fichier dans MQL5/Experts/ via Finder, fais F5 (refresh) dans le panneau Navigator de MT5 — l'EA apparaît
  • Pare-feu macOS qui bloque : Préférences Système → Sécurité & Confidentialité → Pare-feu → vérifier que MT5 n'est pas bloqué pour les connexions sortantes

01Générer ton API key

L'EA s'authentifie auprès de Scalping Radar avec une API key personnelle. Pour la générer :

  • Va sur Settings → Auto-exec MT5
  • Clique "Générer mon API key"
  • Copie immédiatement la valeur affichée — elle ne sera plus visible après fermeture (par sécurité). Si tu la perds, tu pourras en régénérer une nouvelle, mais l'ancienne sera invalidée.

02Télécharger l'EA

Toujours sur Settings → Auto-exec MT5, clique "Télécharger ScalpingRadarEA.mq5". Tu récupères un fichier source (~50 Ko) que tu vas compiler à l'étape suivante.

Place-le dans le dossier Experts de MT5 :

  • Dans MT5 : menu File → Open Data Folder
  • Une fenêtre s'ouvre — navigue dans MQL5/Experts/
  • Glisse ScalpingRadarEA.mq5 dans ce dossier

03Compiler dans MetaEditor

MT5 lit du .ex5 compilé, pas du .mq5. La compilation prend 1 seconde :

  • Dans MT5, ouvre MetaEditor (menu Tools, ou raccourci F4)
  • Dans Navigator → Experts → double-clic sur ScalpingRadarEA.mq5
  • Clique Compile (ou F7)
  • Tu dois voir 0 errors, 0 warnings en bas

Un fichier ScalpingRadarEA.ex5 est généré à côté du source.

04Autoriser les requêtes HTTP

Par défaut, MT5 bloque toute requête réseau émise par un EA. Il faut explicitement whitelister l'URL de Scalping Radar :

  • Tools → Options → Expert Advisors
  • Coche Allow WebRequest for listed URL
  • Dans la zone de texte, ajoute https://app.scalping-radar.online (sans slash final, exactement comme ça)
  • OK
Important
Si tu loupes cette étape, l'EA tourne mais ne reçoit jamais les ordres — tu verras WebRequest 4060 dans les logs.

05Attacher l'EA à un chart

  • De retour dans MT5, F5 ou clic droit "Refresh" sur le panel Navigator → Expert Advisors → tu vois ScalpingRadarEA
  • Drag l'EA sur n'importe quel chart (peu importe la pair, l'EA tourne en background pour tous les ordres reçus)
  • La popup d'inputs s'ouvre. Renseigne :
    • InpApiKey : ta clé copiée à l'étape 1
    • InpServerUrl : https://app.scalping-radar.online (laisse défaut)
    • InpPollingIntervalSec : 30 (peut descendre à 10 si tu veux plus de réactivité)
    • InpDefaultLot : 0.01 (V1 size fixe)
    • InpMagicNumber : laisse défaut (identifie tes trades EA dans History)
    • InpDeviationPoints : 20 (slippage max accepté)
    • InpDryRun : true au premier essai, puis false une fois que tu as vu un cycle complet
  • Onglet Common → coche Allow Algo Trading + Allow live trading
  • OK

Active enfin AutoTrading dans la barre d'outils MT5 (ou Ctrl+E). Le bouton doit passer vert. C'est le master-switch global — sans lui aucun EA ne peut trader.

06Vérifier que ça tourne

Onglet Experts en bas de MT5 (à côté de Trade, History, etc.) — tu dois voir des logs type : 

ScalpingRadarEA initialized — server=https://app.scalping-radar.online polling=30s default_lot=0.01 magic=20260429 dry_run=true

Toutes les 10 minutes :

ScalpingRadarEA alive — polls=20 exec=0 fail=0

Quand un ordre est exécuté :

ScalpingRadarEA order_id=42 EXECUTED ticket=123456789 EURUSD buy

Côté Scalping Radar, sur Settings → Auto-exec MT5, l'indicateur Statut EA doit passer vert · Actif dans les 30 secondes après lancement (heartbeat implicite à chaque poll).

Premier cycle réussi ?
Repasse InpDryRun à false dans les inputs de l'EA (clic droit sur le chart → Expert Advisors → Properties), puis active Auto-exec ON côté Scalping Radar. À partir de là, les setups éligibles passent automatiquement sur ton compte démo.

07Dépannage

"WebRequest 4060" / "URL non whitelistée"

  • Étape 4 manquée. Tools → Options → Expert Advisors → ajoute exactement https://app.scalping-radar.online à la liste

"HttpGet 401 — api_key invalide"

  • Vérifie que tu as collé la clé entière (≥ 16 caractères, sans espace ni saut de ligne)
  • Si tu as régénéré ta clé après l'avoir saisie dans l'EA : remets la nouvelle. L'ancienne est invalidée immédiatement après régénération

"HttpGet 403 — Premium tier requis"

  • Ton compte n'est pas Premium. Subscribe via /pricing

L'EA tourne mais 0 ordre depuis 1h

  • C'est normalement normal — le radar génère un setup éligible quand le scoring déclenche, pas en continu
  • Vérifie que Auto-exec est bien ON côté Scalping Radar (Settings → Auto-exec MT5)
  • Vérifie que tes paires watchlist sont activement scannées par Scalping Radar (Cockpit)

"OrderSend FAILED retcode=10016 INVALID_STOPS"

  • Le SL ou TP est trop proche du prix actuel (slippage de timing). Scalping Radar filtre déjà ce cas en amont, mais le marché peut bouger entre génération et exécution. Skip et attends le prochain setup — pas d'action requise

L'EA est attaché mais le statut Settings reste "Jamais vu"

  • Vérifie que AutoTrading global est vert (Ctrl+E dans MT5)
  • Vérifie que la coche "Allow Algo Trading" est cochée dans les properties de l'EA (clic droit sur le chart → Expert Advisors → Properties → Common)
  • Regarde l'onglet Experts : si tu n'as aucun log "ScalpingRadarEA initialized", l'EA n'a pas démarré

Autre problème ?

Écris à support@scalping-radar.com avec une capture des logs Experts et une description du comportement. Réponse sous 24h en semaine.

Sécurité

  • L'API key transite uniquement en HTTPS
  • L'EA ne place que des ordres au lot fixe InpDefaultLot (0.01 par défaut). Pas de levier supplémentaire ni de pyramiding
  • L'EA ignore les positions ouvertes manuellement (magic_number différent → invisibles pour l'EA)
  • Tu peux désactiver AutoTrading dans MT5 à tout moment (kill switch côté terminal)
  • Côté Scalping Radar, le toggle Auto-exec OFF coupe l'enqueuing sans rien toucher à ton MT5 (kill switch côté serveur)
